# California Demographic Analysis

## Overview

This project analyzes population and median income patterns across California using the California Housing dataset. The goal was to identify regions that combine high population concentrations with higher median income levels.

## Tools

- R
- ggplot2
- dplyr
- maps

## Data Source

California Housing Dataset:
https://www.kaggle.com/datasets/harrywang/housing

## Key Findings

- Los Angeles
- San Francisco Bay Area
- Sacramento

showed strong combinations of population concentration and median income.
